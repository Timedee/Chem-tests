3.1 — The Mole Concept 3.1

Chemists can't count atoms one by one. Instead, they group them into moles — just like buying eggs by the dozen.


Avogadro's Constant:

Nₐ = 6.02 × 10²³ particles/mol

A mole is the amount of substance containing the same number of particles as exactly 12 g of Carbon-12. This means the atomic mass in amu equals the molar mass in grams.


Key Triangle — Particles ↔ Moles:

# = n × Nₐ     (particles → moles: divide by Nₐ)
n = # ÷ Nₐ     (moles → particles: multiply by Nₐ)

Key Triangle — Mass ↔ Moles:

m = n × MM     (moles → grams)
n = m ÷ MM     (grams → moles)

Worked Examples:

  • 0.67 mol CCl₄ → # molecules = 0.67 × 6.02×10²³ = 4.0×10²³ molecules
  • Since CCl₄ has 4 Cl atoms → 4 × 4.0×10²³ = 1.6×10²⁴ Cl atoms
  • 2.0×10²² S atoms → n = 2.0×10²² ÷ 6.02×10²³ = 0.033 mol
  • 53.0 g Na₂CO₃ (MM=106.0) → n = 53.0÷106.0 = 0.500 mol
  • 0.075 mol Fe(NO₂)₃ (MM=193.8) → m = 0.075×193.8 = 15 g

3.2 — % Composition, Empirical & Molecular Formulas 3.2

Percentage Composition: The mass contributed by each element in a compound.

% element = (mass of element / formula mass) × 100%

Example — Mg(OH)₂ (MM = 58.3 g/mol):

Mg: 24.3/58.3 = 41.7%  |  O: 32.0/58.3 = 54.9%  |  H: 2.02/58.3 = 3.46%

To find mass of element in sample: mass of sample × (element mass / formula mass)


Empirical Formula (EF): Simplest whole-number ratio of atoms in a compound.

  • Convert % → assume 100 g → get grams of each element
  • Divide each mass by its molar mass to get moles
  • Divide all moles by the smallest → get ratio
  • Multiply if needed to get whole numbers

Example: 52.2% C, 13.0% H, 34.8% O → C: 4.35/2.18=2, H: 13.0/2.18=6, O: 1 → C₂H₆O


Molecular Formula (MF): Actual number of atoms.

MF = EF × a     where a = Molecular Mass ÷ EF mass

Example: EF = CH (MM=13 u), molecular mass = 78 u → a = 78/13 = 6 → C₆H₆


Hydrates: A hydrate loses water when heated → anhydrous salt.

Mass of water lost = mass of hydrate − mass of anhydrous salt
Then find moles of both → divide to get ratio (x)

Example: 5.03g Cu(NO₃)₂·xH₂O → 3.89g anhydrous. Water lost = 1.14g. n(salt)=0.0207, n(H₂O)=0.0633. Ratio = 3 → Cu(NO₃)₂·3H₂O

3.3 — Stoichiometry & % Yield 3.3

Stoichiometry = the study of relative quantities of reactants and products.


In any balanced equation:

  • Number of atoms and mass ARE conserved ✓
  • Number of moles and molecules are NOT necessarily conserved ✗

Mole Ratio Method (3-Step):

  • Convert mass of given → moles (n = m/MM)
  • Use mole ratio from balanced equation to find moles of wanted
  • Convert moles of wanted → mass (m = n × MM)
Example: N₂ + 3H₂ → 2NH₃
3.4g N₂ → n = 3.4/28.0 = 0.121 mol N₂
Ratio: 1 N₂ = 2 NH₃ → 0.121 × 2 = 0.242 mol NH₃
m = 0.242 × 17.0 = 4.1 g NH₃

Percent Yield:

% Yield = (measured value / expected value) × 100%

Percent Error:

% Error = |measured − expected| / expected × 100%

Example: got 3.5g instead of 4.1g → % Yield = 85%, % Error = 15%

3.4 — Limiting Reagents 3.4

The limiting reagent is the reactant that runs out first and stops the reaction. The other reactant is in excess.


Steps to Solve:

  • Write and balance the chemical equation
  • Convert all given masses to moles (n = m/MM)
  • Divide each mole value by its coefficient in the equation
  • The reactant with the SMALLER result is the limiting reagent
  • Use limiting reagent's moles + mole ratio → find moles of product
  • Convert moles of product → mass

Example 1: Na₂CO₃ + CaCl₂ → CaCO₃ + 2NaCl

34.7g Na₂CO₃: n = 34.7/106.0 = 0.327 mol (÷1 = 0.327)
41.2g CaCl₂: n = 41.2/111.0 = 0.371 mol (÷1 = 0.371)
Limiting = Na₂CO₃ (0.327 smaller)
NaCl: 1 Na₂CO₃ : 2 NaCl → 0.327 × 2 = 0.654 mol
m = 0.654 × 58.5 = 38.3 g NaCl

Example 2: 2Al(OH)₃ + 3H₂SO₄ → Al₂(SO₄)₃ + 6H₂O

21.0g Al(OH)₃: n = 21.0/78.0 = 0.269 mol (÷2 = 0.134)
40.7g H₂SO₄: n = 40.7/98.1 = 0.415 mol (÷3 = 0.138)
Limiting = Al(OH)₃ (0.134 smaller)
H₂O: 2 Al(OH)₃ : 6 H₂O → 0.269 × 3 = 0.807 mol
m = 0.807 × 18.0 = 14.5 g H₂O
